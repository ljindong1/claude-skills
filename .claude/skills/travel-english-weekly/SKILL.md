# Travel English Weekly 학습 스킬

초중급자를 위한 **해외여행 실전 영어 회화** 주간 학습 시스템.
매주 Claude가 여행 상황을 자동 설정하고, 매일 아침 5~10분 학습 콘텐츠를 제공한다.

---

## 핵심 원칙

- **한 상황 집중**: 한 주 내내 같은 상황(예: 호텔)만 다룬다. 월~금 5일이 모두 같은 장소, 같은 맥락 안에서 진행됨
- **깊이 있는 반복**: 월요일 기초 표현 → 화요일 대화문 → 수요일 응용/실수교정 → 목요일 롤플레이 → 금요일 완전 자립 테스트
- **상황 자동 설정**: 사용자가 지정하지 않으면 Claude가 주차별로 상황을 자동 배정
- **초중급 맞춤**: 짧고 실용적인 문장 위주, 한국어 해석 항상 병기
- **아침 루틴**: 매일 아침 5~10분이면 충분한 분량

---

## 주차별 상황 로테이션 (자동 순환)

사용자가 상황을 지정하지 않으면 아래 순서대로 매주 자동 배정한다.

| 주차 | 상황 | 핵심 장소 |
|---|---|---|
| 1주 | ✈️ 공항 & 비행기 | 체크인, 보안검사, 탑승, 기내 |
| 2주 | 🏨 호텔 | 체크인/아웃, 룸서비스, 불편사항 |
| 3주 | 🍽️ 레스토랑 & 카페 | 주문, 알레르기, 계산, 팁 |
| 4주 | 🚇 교통 | 지하철, 버스, 택시, 우버 |
| 5주 | 🛍️ 쇼핑 | 옷 가게, 면세점, 환불/교환 |
| 6주 | 🗺️ 관광 & 길 찾기 | 길 묻기, 티켓 구매, 박물관 |
| 7주 | 🏥 긴급상황 | 병원, 약국, 분실/도난, 경찰 |
| 8주 | 💬 일상 대화 | 날씨, 인사, 소개, 칭찬 |

8주 이후 1주차로 돌아와 심화 버전으로 반복.

---

## 주간 시나리오 시작 포맷

사용자가 "이번 주 영어 학습 시작" 또는 "새 주 시작"을 요청하면:

```markdown
## 🌍 이번 주 여행 영어: [상황명]

### 📍 이번 주 배경 상황
[구체적인 여행 시나리오 — 2~3줄]
예: "뉴욕 JFK 공항에 도착했습니다. 수하물을 찾고 입국심사를 통과해야 합니다.
    영어로 어떻게 말할까요?"

### 🎯 이번 주 목표
이번 주가 끝나면 이런 상황에서 혼자 말할 수 있게 됩니다:
- [목표 1]
- [목표 2]
- [목표 3]

### 📅 이번 주 일정 미리보기 (5일 내내 같은 상황!)
- 월: [상황] 기초 — 핵심 표현 5개 외우기
- 화: [상황] 심화 — 실전 대화문 & 패턴 익히기
- 수: [상황] 교정 — 콩글리시 vs 원어민 표현 비교
- 목: [상황] 실전 — Claude와 롤플레이 연습
- 금: [상황] 완성 — 복습 퀴즈 & 혼자 말하기 테스트
```

> 💡 **왜 같은 상황을 5일간 반복하나요?**
> 월요일에 배운 표현을 금요일엔 아무 도움 없이 자연스럽게 말할 수 있도록
> 같은 맥락 안에서 매일 조금씩 더 깊이 파고듭니다.

## 일별 아침 브리핑 포맷

사용자가 "오늘 영어 공부", "아침 브리핑", "오늘 학습 시작" 등을 요청하면:

### 월요일 — 기초 표현 (같은 상황 1일차)

```markdown
## ☀️ 월요일 아침 영어 — [상황] 1일차: 기초 표현

**⏱️ 오늘 예상 시간**: 7분
**🎯 이번 주 상황**: [상황 한 줄 설명 — 이 상황이 이번 주 내내 계속됩니다]
**📌 오늘의 목표**: 이 상황에서 가장 많이 쓰는 표현 5개 익히기

---

### 📖 오늘의 핵심 표현 5가지

| 영어 | 한국어 | 발음 팁 |
|---|---|---|
| ... | ... | ... |

### 💡 오늘의 문화 팁
[해당 상황의 에티켓 또는 배경 지식 1가지]

### ✅ 오늘의 미션
소리 내어 5번 읽기: "[오늘 가장 중요한 문장 1개]"

> 내일(화요일)엔 오늘 표현들이 실제 대화에서 어떻게 쓰이는지 배웁니다!
```

### 화요일 — 실전 대화문 (같은 상황 2일차)

```markdown
## ☀️ 화요일 아침 영어 — [상황] 2일차: 실전 대화

**📌 오늘의 목표**: 어제 배운 표현들이 실제 대화에서 어떻게 흘러가는지 익히기

### 🎭 오늘의 대화문
(어제 배운 표현이 대화 속에 자연스럽게 등장합니다)

**상황: [구체적 장면]**
> A: ...
> B: ...
> A: ...

*(한국어 해석)*
> A: ...
> B: ...

### 🔑 오늘의 핵심 패턴
**"[패턴]" = [뜻]**
- [활용 예 1]
- [활용 예 2]
- [활용 예 3]

### ✅ 오늘의 미션
A 역할로 대화문 소리 내어 읽기

> 내일(수요일)엔 한국인이 자주 틀리는 표현을 교정합니다!
```

### 수요일 — 응용 & 교정 (같은 상황 3일차)

```markdown
## ☀️ 수요일 아침 영어 — [상황] 3일차: 콩글리시 교정

**📌 오늘의 목표**: 한국인이 이 상황에서 자주 하는 실수 고치기

### ⚠️ 이런 상황엔 이렇게!

❌ 콩글리시: "..."
✅ 원어민 표현: "..."
💬 이유: ...

❌ 콩글리시: "..."
✅ 원어민 표현: "..."
💬 이유: ...

### 🔄 바꿔 말하기 연습
기본 문장을 상황에 맞게 변형해보세요:

기본: "[월~화에 배운 핵심 문장]"
→ [변형 1]: "..."
→ [변형 2]: "..."
→ [변형 3]: "..."

### ✅ 오늘의 미션
변형 문장 3개 직접 만들어보기

> 내일(목요일)엔 Claude와 실제로 대화 연습을 합니다!
```

### 목요일 — 롤플레이 (같은 상황 4일차)

```markdown
## ☀️ 목요일 아침 영어 — [상황] 4일차: 실전 롤플레이

**📌 오늘의 목표**: 월~수 3일간 배운 것을 실제 대화로 써먹기

### 🎮 오늘의 롤플레이

Claude가 [역할(예: 호텔 직원, 점원)]을 맡겠습니다.
당신은 여행자 역할입니다.

**오늘 연습할 장면**: [구체적 장면 — 이번 주 상황 안에서]

제가 먼저 말을 걸겠습니다:
> "[영어 첫 마디]"
> ([한국어 해석])

한국어로 답하셔도 괜찮아요! 영어로 어떻게 말하는지 바로 알려드립니다. 😊

> 내일(금요일)엔 이번 주 배운 것을 모두 혼자 말할 수 있는지 확인합니다!
```

### 금요일 — 완전 자립 테스트 (같은 상황 5일차)

```markdown
## ☀️ 금요일 아침 영어 — [상황] 5일차: 혼자 말하기 테스트

**📌 오늘의 목표**: 아무 도움 없이 이번 주 상황을 영어로 혼자 소화하기

### 📝 이번 주 복습 퀴즈
(이번 주 내내 같은 상황을 배웠습니다. 이제 혼자 말할 수 있나요?)

다음을 영어로 말해보세요:
1. [한국어 문장 — 월요일 기초 표현]
2. [한국어 문장 — 화요일 대화 패턴]
3. [한국어 문장 — 수요일 교정 포인트]
4. [한국어 문장 — 응용 표현]
5. [한국어 문장 — 이번 주 가장 중요한 문장]

<정답>
1. ...  2. ...  3. ...  4. ...  5. ...

### 🎯 최종 미션: 1분 혼자 말하기
아래 상황을 영어로 처음부터 끝까지 혼자 소화해보세요:
[이번 주 상황을 통합한 짧은 시나리오]

### 🏆 이번 주 완료!
[이번 주 배운 핵심 3가지 요약]

**다음 주 예고**: [다음 주 상황] — 준비되셨나요? 😊
```

## 롤플레이 진행 방식

목요일 또는 사용자가 "연습하자", "롤플레이 해줘"라고 하면:

1. Claude가 현지인/직원 역할로 영어로 말을 건다
2. 사용자가 한국어로 답하면 → Claude가 영어 표현을 알려주고 교정
3. 사용자가 영어로 답하면 → Claude가 자연스러운지 피드백
4. 5~7번 주고받으면 한 scene 완료

```
예시 흐름:
Claude: "Good afternoon! Do you have a reservation?"
사용자: "네, 있어요. 홍길동이에요"
Claude: ✅ 좋아요! 이렇게 말하면 돼요:
        "Yes, I have a reservation under the name Hong Gil-dong."
        → "under the name" = 이름으로 예약했을 때 쓰는 표현이에요.
        한번 따라해보세요! 😊
```


## 아침 예약 학습 방법

Claude.ai는 자동 예약 발송 기능이 없으므로, 아래 방법을 활용한다:

**추천 방법 — 스마트폰 알림**
1. 매일 아침 원하는 시간에 알람 설정
2. 알람 이름: "영어 공부 시작 🌍"
3. 알람 울리면 Claude 앱 열고 입력: **"오늘 여행 영어 브리핑 시작해줘"**

**Google Calendar 활용**
- 매일 반복 일정 생성
- 설명란에: "Claude에게 '오늘 여행 영어 학습 시작' 메시지 보내기"
